# ICACON 2026 Companion (Web Demo)

Vite + React mobile-web prototype for **ICACON 2026** (Aligarh).

Production target is the native app in `../icacon-companion-mobile`.

## Run

```bash
npm install
npm run dev
```

## Screens

Matches the mobile app 1:1 (mobile is source of truth):

- **Home** — brand hero, programme / navigate / info shortcuts  
- **Navigate** — indoor map placeholder + open venues in Maps  
- **Programme** — workshops list + Day 1/2 PDF programme  
- **Info** — resources, contacts, hosts  

Static client-side demo — no backend.
